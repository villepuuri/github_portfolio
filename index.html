Egg carton detector

In an egg sorting machine was a problem: sometimes it did not put exactly 10 eggs to a carton. The fault could not be found, so I got an idea that I could built a system to count the eggs in every carton, and notify the user, if there is a problem. The system is built on a Raspberry Pi, and it uses OpenCV for processing the data.

  • Raspberry Pi
